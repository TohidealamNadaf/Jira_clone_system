Getting Started

Set up and start developing in 5 minutes

1. Installation

Already installed in XAMPP at:

C:\xampp\htdocs\jira_clone_system\

Access the application:

http://localhost/jira_clone_system/public/

Code Standards & Conventions

How to write code for this project

Authority: All standards are documented in AGENTS.md - refer to it as your development bible.
PHP Conventions
  • Version: PHP 8.2+ with strict types
  • Strict Types: declare(strict_types=1); on all files
  • Namespaces: App\{folder}\ClassName
  • Type Hints: All parameters and return types required
  • Naming: Classes: PascalCase, methods/properties: camelCase

Example:

declare(strict_types=1);

namespace App\Services;

class UserService {
    public function getUserById(int $id): ?array {
        // Implementation
    }
}
Database Queries

Always use prepared statements with PDO:

// SELECT
$users = Database::select('SELECT * FROM users WHERE id = ?', [$id]);

// INSERT
$success = Database::insert('users', ['name' => $name, 'email' => $email]);

// UPDATE
$success = Database::update('users', ['name' => 'John'], 'id = ?', [1]);

// DELETE
$success = Database::delete('users', 'id = ?', [1]);
Controllers & Views

Controllers extend App\Core\Controller:

class UserController extends Controller {
    public function index(): string {
        $users = UserService::all();
        return $this->view('users.index', ['users' => $users]);
    }
    
    public function store(): void {
        $data = $this->request->validate(['email' => 'required|email']);
        UserService::create($data);
        $this->redirect('/users');
    }
}

Views use PHP short tags:

<?= csrf_token() ?>
<?= htmlspecialchars($user['name']) ?>
<?php if ($user['active']): ?>
    Active user
<?php endif; ?>
