===============================================================================
TROUBLESHOOTING
===============================================================================

ISSUE: Design still looks different for different users
FIX: 
  1. Clear browser cache (Ctrl+Shift+Del)
  2. Hard refresh (Ctrl+F5)
  3. Check browser console (F12) for CSS errors
  4. Verify design-consistency.css is linked in app.php

ISSUE: Colors are still blue, not plum
FIX:
  1. Check CSS variables in app.php (lines 39-51)
  2. Verify --jira-blue = #8B1956
  3. Clear cache and refresh
  4. Check if another CSS is overriding (z-index issue)

ISSUE: Layout is still cramped
FIX:
  1. Check main content max-width in app.php (should be 1400px)
  2. Check padding (should be 32px)
  3. Verify design-consistency.css is loaded (F12 → Network)
  4. Check for competing CSS in other stylesheets

ISSUE: Responsive not working
FIX:
  1. Verify viewport meta tag exists
  2. Check media queries in design-consistency.css
  3. Test in browser DevTools device emulation
  4. Hard refresh and try again
