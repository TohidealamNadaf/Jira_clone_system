DEPLOYMENT STEPS
================

1. Clear Browser Cache
   - Press: CTRL + SHIFT + DEL
   - Select "All time"
   - Clear all options
   - Click "Clear data"

2. Hard Refresh
   - Press: CTRL + F5
   - Or: CTRL + SHIFT + R (some browsers)
   - Wait for page to fully load

3. Navigate to an Issue
   - Go to: /projects/CWAYS/issues
   - Click on any issue (e.g., CWAYS-4)
   - Scroll down to see "Attachments" section

4. Test Attachments
   If issue has no attachments:
   - See "No attachments yet..." message
   - This is the empty state (working correctly)

   If issue has attachments:
   - See grid of attachment cards
   - Each card shows:
     * File icon (colored circle)
     * File name
     * File size (e.g., "2.5 MB")
     * Upload time (e.g., "2 hours ago")
     * Download button (arrow icon)
   - Click download to test

TESTING WITH ATTACHMENTS
