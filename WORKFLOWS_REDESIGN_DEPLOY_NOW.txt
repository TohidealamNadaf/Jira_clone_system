COMMON ISSUES & SOLUTIONS
================================================================================

ISSUE 1: Page looks broken/unstyled
  SOLUTION: 
    1. Clear cache (CTRL+SHIFT+DEL)
    2. Hard refresh (CTRL+F5)
    3. Wait 5 seconds for CSS to load
    4. If still broken, check browser console (F12)

ISSUE 2: Buttons don't work
  SOLUTION:
    1. Check browser console (F12) for errors
    2. Verify JavaScript is enabled
    3. Check CSRF token is present
    4. Try different browser

ISSUE 3: Modal won't close
  SOLUTION:
    1. Press ESC key
    2. Click outside modal
    3. Refresh page (F5)
    4. Check browser console

ISSUE 4: Sidebar missing on desktop
  SOLUTION:
    1. Resize window (may be in mobile view)
    2. Check browser zoom (should be 100%)
    3. Verify CSS loaded (check Sources in DevTools)
    4. Hard refresh

ISSUE 5: Text looks wrong size/weight
  SOLUTION:
    1. Check system fonts available
    2. Clear cache completely
    3. Hard refresh
    4. Try different browser

Still having issues? Check documentation files.
